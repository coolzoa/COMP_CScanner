El siguiente programa es un preprocesador y  un scanner del lenguaje C (escrito en C). El programa toma como entrada un archivo c (debe estar en el mismo directorio que el programa).

Para usarlo se llama por terminal ./programa1 nombreArchivo.c

En caso de que el programa lea el archivo .c se creara un pdf beamer con los tokens que reconocio el scanner con diferentes colores y graficos.
